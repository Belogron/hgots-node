Welcome to the Administration Interface

This is your home for managing who has access to this system.

  • All the users that have access to this system are listed in the list.
  • You can create new users and edit existing ones in the list.
  • There are three different user types:
    1. Admin: Has full rights. He can create new users, controllers and admins. Can open the door.
    2. Controller: Has rights to create new users. Can open the door.
    3. User: Can open the door.

If you have any questions, see the About page.
